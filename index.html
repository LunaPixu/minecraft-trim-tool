Luna Pixu's Minecraft Trim Generator

Twitter: @Luna_Pixu, Github: LunaPixu

This is a really crude Minecraft tool for generating simple armor trim materials for the upcoming Minecraft Java Edition update 1.20 (or the upcoming 1.19.4 via experimental datapacks).
Simply input the name of your trim material (and its corresponding item), pick a color for the item tooltip, and specify the model index. Once done, you'll be given a basic resource pack and data pack combination.

Note: You will have to edit your packs further in order for them to work fully. e.g. I don't support adding the trim palette texture or palette overrides (like diamond trim on diamond armor).
This tool exists just so you don't have to constantly copy-paste a ton of .json files.
